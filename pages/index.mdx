# Why Submodule?
 
Submodule is the simpliest possible IOC using function. As the currency in Submodule are all functions, Submodule gives a lot of flexibility and reusability to the library users

## 1 minute example
<Steps>
### Install dependencies
```bash copy
npm install @submodule/core chalk
```

### Write your color app
```javascript filename="main.mjs" copy
import { prepareExecutable } from "@submodule/core"
import chalk from "chalk"

const { execute } = prepareExecutable({
  createConfig() {
    const color = chalk[process.env.LOG_COLOR] || chalk.green

    return { color }  
  },
  createServices({ config }) {
    return {
      log(...args) {
        console.log(...args.map(arg => config.color(arg)))
      }
    }
  }
})

execute(( { services } ) => {
  services.log('hello', 'world')
})
```

### Execute code
```bash copy
node main.mjs # should return hello world in green
LOG_COLOR=red node main.mjs # should return hello world in red
```

</Steps>
