# Typescript support

Submodule comes with tight typescript integration. There are few quality of life factor for your integration

## Auto-inferring types
Submodule function supports type inteference in the submodule definition

```typescript {5, 7}
prepareExecutable({
  createConfig() {
    const color = chalk[process.env.LOG_COLOR] || chalk.green

    return { color }  
  },
  createServices({ config }) {
    //             ^? { color: ...chalk function}
    return {
      log(...args) {
        console.log(...args.map(arg => config.color(arg)))
      }
    }
  }
})

```
