Concepts
Type shapes

Most pieces of Submodule requires a shape. A shape reflects how the application is designed and used. There are likely few types of Shapes

Configuration is an object that holds application configuration. It reflected using the typescript code indicating what configuration looks like Configuration creation function is the function that responsible to load configuration from different sources and making sure configuration in write shape

type Config = {
  mongoUrl: string 
  username?: string 
  password?: string 
}
 
function createConfig(): Config {
  return {
    mongoUrl: process.env.MONGO_URL || 'mongodb://mongodb0.example.com:27017',
    username: process.env.MONGO_USER,
    password: process.env.MONGO_PASSWORD
  }
}

If you are not font of process.env, you can quickly swap with with .env or .yaml without losing type comprehension
